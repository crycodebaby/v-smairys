## Aktuelle strategische Eckpfeiler (Kurzfassung)

- **Primärangebot:** premium, individuelle Unternehmenswebsites / Webdesign.
  SEO & Google Ads sind **sekundäre Add-ons**.
- **Zielkunden:** Handwerk, Immobilien, Gastronomie im Saarland / Saarbrücken.
- **Primäre Conversion:** Erstgespräch (Telefon **oder** Google-Calendar-Booking).
  Kontaktformular ist sekundär.
- **Pakete (Goldilocks):** Entry ~1.200 € · Mitte ~2.600–2.800 € (empfohlen) ·
  Premium ~3.600 €.
- **Positionierung:** nicht der billigste Anbieter; Website als Vertriebs- und
  Vertrauens-Asset.
- **Brand:** brutalist/luxuriös, Black/Off-White + zurückhaltender Kupfer-/
  Amber-Akzent.
